# Online Classroom Portal

## Description

The **Online Classroom Portal** is a web-based application designed to simplify online teaching and learning. It provides a platform where teachers can manage classes, upload study materials, post assignments, and communicate with students. Students can log in to access course content, submit assignments, and stay updated with classroom activities. The system offers an organized and user-friendly interface, making the learning process more efficient and accessible.

## Features

* Secure Student and Teacher Login
* Course and Classroom Management
* Upload and Download Study Materials
* Assignment Creation and Submission
* Announcements and Notifications
* User-Friendly Interface

## Technologies Used

* HTML
* CSS
* JavaScript
* PHP
* MySQL

## Purpose

The main objective of this project is to provide a simple and efficient online platform that enhances communication between teachers and students while supporting digital learning and classroom management.
